CKeditor-yaspeller

This plugin for CKEditor allows "on the fly" spell-checking for Russian & English languages. Check is done via Yandex API, and it's absolutely free. That's a good alternative to costy official Scayt plugin. Misspelled words are marked, suggestions can be selected from context menu.

Demo

Mark errors as:

  • red font
  • blue font and yellow background
  • black font and red underline
